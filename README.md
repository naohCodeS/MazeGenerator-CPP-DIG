## 理由
以前壁伸ばし法で同じアルゴリズムを書いたのに、また書いた理由。  
迷路生成を３次元に拡張しようとしたときに、壁伸ばし法ではうまくいかないから。  

## 感想
以前に壁伸ばし法で迷路生成アルゴリズムを書いたことがあるが、今回の穴掘り法のほうがアルゴリズムがシンプルで実装もミスりずらく楽に感じた。
